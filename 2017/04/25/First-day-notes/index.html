###复习数组中的方法

  • push()
  • unshift()
    • 可以在数组开始添加元素,要加多少项就加多少个参数
  • pop()
  • shift()
  • slice(n, m)
  • splice(n, m, X1..)
  • concat()
  • indexOf(), lastIndexOf()
    • 都可以接受两个参数
    • 第一个是要查找的子字符串
    • 第二个可选参数是偏移量(从第几项开始查找)
  • forEach(function(item, index, input){})
  • map(function(item, index, input){})
  • join(“”)
    • 将数组元素拼接成字符串
    • 参数是分割符
    • 如果参数是空字符串,那么返回的会是数组元素组成的没有任何连接符的字符串.
  • toString()
    • 返回值跟数组将方括号换成引号一样的效果(以引号中内容为值的字符串)
  • reverse()
  • sort(function(a, b){return a - b})
  • map(function(tm, ndx, npt){return tm})
  • forEach(function(tm, ndx, npt){})
  • map(function(tm, ndx, npt){})
